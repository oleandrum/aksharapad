Devanagari IME · Sanskrit

Type Sanskrit, get Devanagari.

A Devanagari input method for the browser. Type in IAST, Harvard-Kyoto, ITRANS, or Velthuis — press space, and the word converts to Devanagari instantly. Powered by the real Aksharamukha engine, running entirely offline in your browser.

Open the composer Download for macOS

On iPhone/iPad: open the composer, then Share → Add to Home Screen for an installable, offline-capable icon.

How it works

Four keyboard schemes

IAST, Harvard-Kyoto, ITRANS, and Velthuis — switch anytime, mid-sentence.

Space-triggered conversion

Type a word, hit space, and it becomes Devanagari immediately — no separate "convert" step.

Vedic accent marks

A palette of the full Unicode Vedic Extensions block for svara marks, anusvara variants, and more.

Repository Releases Aksharamukha (original) Official IME (100+ scripts)

This is a personal, reduced-scope interface limited to four schemes. For the complete tool supporting many more scripts and languages, visit the official Aksharamukha IME linked above.
